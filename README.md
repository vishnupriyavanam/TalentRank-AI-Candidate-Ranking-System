# TalentRank AI Candidate Ranking System

 Overview

TalentRank AI is an intelligent candidate ranking system developed for the INDIA RUNS Hackathon. The system analyzes candidate profiles and ranks them based on relevance to a given job description.

 Features

* Candidate ranking based on job fit
* Explainable scoring and reasoning
* Top 100 candidate recommendations
* Automated submission generation

 Technologies Used

* Python
* Pandas
* NumPy

 Files

* rank.py – Candidate ranking logic
* final_submission.csv – Final ranked candidates
* candidate_schema.json – Dataset schema
* validate_submission.py – Submission validation

 Approach

1. Load candidate dataset
2. Extract relevant features
3. Calculate candidate scores
4. Rank candidates by relevance
5. Generate top 100 recommendations

 Output

The system produces a ranked list of the top 100 candidates along with scores and reasoning for each recommendation.
